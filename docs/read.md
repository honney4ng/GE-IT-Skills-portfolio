
Objective: Condense complex data into a scannable infographic report using a structured academic framework.

Instructions:

Select a real-world problem in your discipline (the "Burning Problem").

Create a vertical infographic in Canva that outlines your solution using the 5-Chapter Framework:

Chapter 1: Introduction (The problem statement).

Chapter 2: Literature Review (What current solutions exist?).

Chapter 3: Methodology (Your step-by-step plan).

Chapter 4: Implementation/Results (Data visualizations or charts supporting your idea).

Chapter 5: Conclusion (The final impact).

Download the infographic as a high-quality PNG or PDF.

Upload it to your GitHub repository inside a folder named /docs
